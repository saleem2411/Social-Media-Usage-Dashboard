# Social-Media-Usage-Dashboard
# Project Objective
The objective of this project is to analyze social media usage patterns and daily screen time behavior across different age groups and demographics. The dashboard aims to provide actionable insights into how users interact with major social media platforms, how much time they spend online, and how engagement varies by age and gender.
# Dataset Used
- <a href="https://github.com/saleem2411/Social-Media-Usage-Dashboard/blob/main/Mental_Health_and_Social_Media_Balance_Dataset.csv">Dataset</a>
# KPIS
- What is the average daily screen time across all users?
- How many total users are using Social Media?

# Process
- Gather raw social media usage data from Kaggle.com
- Clean data by removing duplicates, handling missing values.
- Defined key performance indicators (KPIs): average screen timet, Total users.
- Donut chart for days without social media by gender.
- Line chart for no of users by age.
- Column chart for total users by social media.
- Bar chart for daily  screen time by age in hours.
- Slicer filters for gender and age.
- Applied DAX measures for dynamic calculations and accurate insights.

# Project Insights
- TikTok, X (Twitter), and LinkedIn appear to be the most used platforms, each showing user counts close to 90–100.
- The average screen time is 5.53 hours, indicating a relatively high level of daily digital engagement.
- Several age groups (particularly 35–40 and 45+ range) show screen time values above the average, suggesting increased engagement or dependency.
- User counts remain fairly consistent across ages 16–50, with small spikes around age 26, 32, and 45, suggesting these ages are slightly more active.

# Final Conclusion
The dashboard shows that social media usage is consistently high across all age groups, with an average screen time of 5.53 hours. Platform engagement is evenly distributed, and gender-based detox patterns are balanced. Overall, users of all demographics remain highly active on digital platforms, indicating strong and steady social media engagement.
